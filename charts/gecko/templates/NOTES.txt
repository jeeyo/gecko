Access the dashboard with:

  kubectl port-forward -n {{ .Release.Namespace }} \
    svc/{{ include "gecko.fullname" . }}-web 8080:{{ .Values.web.port }}

Then open http://localhost:8080/

The API is available at:
  kubectl port-forward -n {{ .Release.Namespace }} \
    svc/{{ include "gecko.fullname" . }}-api 3001:{{ .Values.api.port }}

NOTE: Without Ingress the SPA's /api requests won't proxy to the API
automatically. Either enable ingress or use an additional reverse proxy.
